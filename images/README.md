# Bilder vom Bauprozess

Hier werden alle Fotos vom Bauprozess gesammelt.

## Ordnerstruktur

Bitte Bilder mit folgendem Namensformat hochladen:
- `YYYY-MM-DD-beschreibung.jpg`
- Beispiel: `2026-02-01-erste-teile.jpg`

## Kategorien

- **planning/** - Skizzen und Planungsfotos
- **assembly/** - Zusammenbau-Fotos
- **testing/** - Test- und Debugging-Fotos
- **final/** - Fotos vom fertigen Projekt

_Bilder werden während des Projekts hinzugefügt_
