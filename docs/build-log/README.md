## Einträge

### Template für neue Einträge

Kopiere diese Vorlage für jeden neuen Eintrag:

```markdown
## [Datum: YYYY-MM-DD] - Titel

### Zusammenfassung
Kurze Beschreibung was heute gemacht wurde.

### Details
- Aufgabe 1
- Aufgabe 2
- Aufgabe 3

### Herausforderungen
Welche Probleme sind aufgetreten?

### Gelernt
Was habe ich heute gelernt?

### Fotos
![Beschreibung](../../images/YYYY-MM-DD-beschreibung.jpg)

### Nächste Schritte
- [ ] Aufgabe 1
- [ ] Aufgabe 2
```
