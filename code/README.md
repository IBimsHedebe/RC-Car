# Code und Firmware

Falls das RC Car programmierbare Komponenten enthält (z.B. Arduino, ESP32), wird der Code hier gespeichert.

## Struktur

```
code/
├── firmware/          # Firmware für Mikrocontroller
├── calibration/       # Kalibrierungs-Scripts
└── testing/           # Test-Code
```

## Plattformen

_Wird aktualisiert basierend auf den verwendeten Komponenten_

## Abhängigkeiten

_Benötigte Libraries und Tools werden hier dokumentiert_
